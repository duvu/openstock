# Phased roadmap (deprecated)

The R0–R14 phase plan has been retired as an active roadmap. It mixed deployment, TUI, runtime, research, backtest and ML work into a second execution queue that no longer matched completed issues or current data dependencies.

Use GitHub issue [#238 — OpenStock core data and knowledge loop](https://github.com/duvu/openstock/issues/238) for the current sequence and [`ROADMAP.md`](../../ROADMAP.md) for governance.

Stable concepts from the former document remain represented in architecture and capability documentation. Live priority, completion status and dependencies now exist only in GitHub Issues. The historical R0–R14 text remains available in Git history.

Do not restore a phase checklist here. If a missing capability should enter the roadmap:

1. create a focused GitHub issue with dependencies and acceptance criteria;
2. link any OpenSpec requirements/design to that issue;
3. add the issue to #238 after roadmap review.
